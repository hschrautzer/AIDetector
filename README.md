# AIDetector
Realisation of a tool detecting whether a text is generated by AI or by a human. This is a learning project for me 
based on this [blog article](https://medium.com/@tomrodolfolee/building-an-ai-detector-from-scratch-part-i-db72bc2bdadb) by Tom Li.
Its purpose is mainly for me to deepen my skills in deep learning and NLP.
Instead of Tom Li, I use German texts. Maybe I can see whether the model works better for German or for the English
language.

## Data
For the human texts I consider articles from [tagesschau](https://www.tagesschau.de).

### Data Collection
An abstract base class `ICollector` is a blueprint for the article collection classes. The classes
`CTagesschauCollector`,..., implement this interface and save the retrieved and cleaned articles to the `data` folder.
